# Era

----

> Era is a discord bot intended for usage on a community discord server exclusively. 


----
## Features

- Twitch Stream WebHooks
- Quiz
- Word Definitions & Flags
- RNG (Rolling)
- Reddit link swapper (reddit link -> .mp4)
